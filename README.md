## A basic React Calculator which van be used to perform basic mathematical calculations

1. Add
2. Subtract
3. Divide
4. Multiply
5. Clear Screen
6. Backspace
7. Equals 
8. Add/Subtract/Divide/Multiply Clubbed together
9. Two symbols cannot be placed together
